Code Generator II Operations

The Toolbar

Open and Load Database

Opens the database and displays table information.

Select Table To Create

When the database is loaded the drop down combo box is loaded with the name of each table in fhe database. When a table is selected the module is created and both the header (".h") and body (".cpp") file are written. The names are of the files are placed in the save dialog and the path is remembered from the last file that was created.

Create DBtable Module

A dialog box is presented to the user with a check box for each table in the database. The tables for which modules have been created in the current execution of the application are checked.

The DBtable module contains an object for each table selected in the dialog box. C++ initializes global objects in the order that they appear in a file. DBtable pointer to each replica of a database table module. The pointers are installed in DBtable during initialization of the application before the main initInstances is executed. Thus this is am important file to be correct.

Save File

Displays a Save File Dialog. However, it has no purpose in this application.

Print

Will do all the normal things to print the currently displayed sorted list.

Options

The options dialog allows specifying the Display and Printer Font Size Scale Factor, the printer orientation and the printer margins.

About

Displays the About Box. The about box uses text from the resource file (TestApp.rc) for the output information. There is a standard about box dialog in the Library.

Exit

Exit the application with no reservations.
